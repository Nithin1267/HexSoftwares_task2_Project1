# Student Performance Prediction and Model Evaluation

This project predicts whether a student will pass or fail using machine learning.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Files
- `project1_model_evaluation.ipynb` - Main notebook
- `student_data.csv` - Dataset

## Model Evaluation Metrics
- Accuracy
- Precision
- Recall
- Confusion Matrix
- Classification Report
